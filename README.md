<h2 align="center">📌 Projetos em Destaque</h2>

### 🎬 [ChooseMovie](https://github.com/jhownny/ChooseMovie)
**Sistema de recomendação de filmes em Python**
- **O que faz:** Escolhe aleatoriamente um filme e ano de lançamento a partir de uma base de dados (CSV ou MySQL).
- **Tecnologias:** Python, PySimpleGUI, MySQL Connector, Pandas.
- **Aprendizado:** Manipulação de dados, conexão com banco de dados e criação de interfaces gráficas simples.

### 🥩 [PHP-SistematiChuleta](https://github.com/jhownny/PHP-SistematiChuleta)
**Sistema web para churrascaria**
- **O que faz:** Interface web responsiva para gerenciamento e apresentação de produtos de uma churrascaria.
- **Tecnologias:** PHP, Bootstrap, HTML, CSS.
- **Aprendizado:** Desenvolvimento full-stack com PHP no backend e Bootstrap no frontend.
